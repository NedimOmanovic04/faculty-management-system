# Faculty Management System (C#)

A faculty management application developed in C# for managing users and academic data.  
The system provides functionality for user management and database operations.

## Features

- User management system
- Data storage and retrieval
- Database connectivity
- Structured application architecture
- CRUD operations

## Tech Stack

- C#
- .NET Framework / .NET
- SQL Database
- Object-Oriented Programming

  ## What I Learned

- Object-oriented programming in C#
- Database integration in .NET applications
- Application architecture design
- User data management
